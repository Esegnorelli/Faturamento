# Faturamento  Insights (Streamlit)
Dashboard minimalista com KPIs MoM, rankings, alertas e sazonalidade.

## Como rodar
pip install -r requirements.txt
streamlit run Faturamento_app_minimal_insights.py

## Dados
Coloque `Faturamento.csv` na mesma pasta do script. O app tenta `;` e, se falhar, `,`.
